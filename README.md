## Project Description
Functional AI chatbot that answers questions about the Chicago Bears from an existing knowledge base. Useful for those looking to learn more about their local team or already-avid fans wanting to test their knowledge, the web application allows for easy messaging back and forth between a chatbot and human user.

## Features
* Ability to respond to queries about the Chicago Bears, including current players, historical facts, and general information.
* Chatbot with an enthusiastic demeanor, which handles queries it doesn't understand to prevent hallucinations.
* Chat-style frontend complete with avatars, animated loading bubbles, and team colors worked into the design.

## Installation & Usage
1. Clone the repository.
2. Install dependencies (pip3 may be required on some devices):
    * `pip install sympy=1.11.1`
    * `pip install Flask`
    * `pip install flask-cors`
3. Navigate to the project folder.
4. Run the chatbot: `python3 -i app.py`
5. Right click `index.html` and open in browser of choice.
